# Python Refactor Examples

This repository shows how messy or poorly written Python code
can be refactored into clean, readable, and maintainable code.

Each example includes:
- A messy version
- A refactored clean version
- A short explanation of improvements

---

**Problem:** Hard-to-read code with poor naming and structure.  
**Refactor:** Improved naming, structure, and readability.
